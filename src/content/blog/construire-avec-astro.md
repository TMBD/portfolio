Ce portfolio est construit avec **Astro**, **Tailwind CSS** et un peu de
TypeScript. Voici pourquoi cette combinaison fonctionne si bien pour un site de
contenu.

## Rapide par défaut

Astro n'envoie aucun JavaScript sauf si vous le demandez. La majeure partie de ce
site est du HTML statique : les pages se chargent instantanément.

## Le contenu comme donnée

Mes expériences, projets et compétences vivent dans des fichiers de données typés
— une source unique de vérité rendue partout. Ajouter un projet, c'est une ligne.

## Bilingue dès le départ

Le site est entièrement bilingue (français et anglais) avec un sélecteur de
langue, basé sur le routage i18n d'Astro.

Il y a beaucoup à dire sur chacun de ces points — les prochains articles y
reviendront.
