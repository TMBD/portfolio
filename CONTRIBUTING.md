# 🤝 Contribuindo para o Dev Personal Theme

Obrigado por considerar contribuir para este projeto! Aqui estão algumas diretrizes para ajudar você a começar.

## 🚀 Como Contribuir

### 1. Fork o Projeto

- Faça um fork do repositório
- Clone seu fork localmente
- Crie uma branch para sua feature/correção

### 2. Configuração do Ambiente

```bash
git clone https://github.com/seu-usuario/dev-personal-theme.git
cd dev-personal-theme
npm install
npm run dev
```

### 3. Faça suas Alterações

- Mantenha o código limpo e bem documentado
- Siga as convenções de nomenclatura existentes
- Teste suas alterações localmente

### 4. Commit e Push

```bash
git add .
git commit -m "feat: adiciona nova funcionalidade"
git push origin sua-branch
```

### 5. Abra um Pull Request

- Descreva claramente suas alterações
- Inclua screenshots se aplicável
- Referencie issues relacionadas

## 📝 Convenções de Commit

Usamos [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` nova funcionalidade
- `fix:` correção de bug
- `docs:` alterações na documentação
- `style:` formatação, ponto e vírgula, etc
- `refactor:` refatoração de código
- `test:` adição ou correção de testes
- `chore:` tarefas de manutenção

## 🐛 Reportando Bugs

Ao reportar bugs, inclua:

- Descrição clara do problema
- Passos para reproduzir
- Comportamento esperado vs atual
- Screenshots (se aplicável)
- Informações do ambiente (OS, browser, etc)

## 💡 Sugerindo Melhorias

Para sugerir melhorias:

- Descreva a funcionalidade desejada
- Explique por que seria útil
- Forneça exemplos de uso
- Considere a implementação

## 📋 Checklist para Pull Requests

- [ ] Código testado localmente
- [ ] Documentação atualizada (se necessário)
- [ ] Commits seguem as convenções
- [ ] Não há conflitos com a branch main
- [ ] Descrição clara das alterações

## 🎨 Padrões de Código

- Use TypeScript para tipagem
- Siga as convenções do Prettier
- Mantenha componentes pequenos e reutilizáveis
- Use nomes descritivos para variáveis e funções
- Adicione comentários quando necessário
